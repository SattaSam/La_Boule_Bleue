LA BOULE BLEUE — INSTALLATION MOBILE — VERSION 13 FINALE

Cette version est une application web installable (PWA).
Cette édition réinitialise une fois les anciens profils, inventaires et scores afin de repartir de zéro.
Le mode entraînement permet ensuite de choisir librement un niveau, sans modifier l’aventure principale.

IMPORTANT
Une PWA doit être ouverte depuis une adresse HTTPS ou depuis localhost.
Elle ne peut pas être installée directement en ouvrant index.html depuis le ZIP.

MISE À JOUR DU SITE GITHUB
Utilisez le fichier La_Boule_Bleue_GitHub.zip : ses fichiers sont déjà placés à la racine.
1. Décompressez La_Boule_Bleue_GitHub.zip.
2. Dans le dépôt SattaSam/La_Boule_Bleue, remplacez les fichiers situés à la racine par le contenu extrait : index.html, manifest.webmanifest et service-worker.js.
3. Remplacez également le dossier icons. Ne déposez pas un dossier parent La_Boule_Bleue_Mobile dans le dépôt.
4. Attendez une à deux minutes, puis ouvrez :
   https://sattasam.github.io/La_Boule_Bleue/
5. Sur le téléphone, supprimez l’ancienne application installée si elle existe, puis rechargez cette adresse dans le navigateur avant de réinstaller.
6. Le menu d'accueil affiche le bouton « Installer sur ce téléphone ». Le jeu accepte le portrait et le paysage sans verrouillage.

ANDROID / CHROME
1. Décompressez le dossier sur un hébergement HTTPS.
2. Ouvrez index.html dans Chrome.
3. Touchez le menu ⋮ puis « Installer l'application » ou « Ajouter à l'écran d'accueil ».

IPHONE / IPAD / SAFARI
1. Ouvrez l'adresse HTTPS du jeu dans Safari.
2. Touchez Partager.
3. Choisissez « Sur l'écran d'accueil ».

ORDINATEUR
Le fichier index.html reste jouable directement. L'installation PWA complète nécessite également HTTPS ou localhost.
